# UI 控件重设计

## 概述

重新设计左侧栏文件夹的展开/闭合按钮和右侧栏模式选择下拉框，使其与应用的"温暖编辑风格"保持一致。

## 问题

1. **展开/闭合按钮**：有明显的黑底色，颜色不透明，与应用风格不协调
2. **模式选择下拉框**：打开状态有黑底色，样式与应用不一致

## 设计方案

### 一、展开/闭合按钮

**风格：** 三角形指示器（类似 macOS Finder）

**样式：**
- 收起状态：向右三角形 `▶`
- 展开状态：向下三角形 `▼`
- 无背景，透明

**配色：**

| 状态 | 颜色 | 说明 |
|------|------|------|
| 默认 | 棕色 `#8B5A2B` | 与文字颜色协调 |
| hover | 琥珀色 `#D4A574` | 提供交互反馈 |

**实现方式：**
- 修改 `sidebar.py` 中的 `drawBranches()` 方法
- 移除背景圆绘制
- 使用 QPainter 绘制三角形线条

---

### 二、模式选择下拉框

**风格：** 扁平简约

**主体样式：**
- 圆角：6px
- 内边距：4px 8px

**配色：**

| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | 象牙白 `#FFFEF9` | 与输入框统一 |
| 边框 | 淡褐色 `#E8DFD5` | 细边框 |
| hover 背景 | 浅米色 `#FDF8F0` | 交互反馈 |
| 下拉箭头 | 棕色 `#8B5A2B` | 与三角形统一 |
| 文字 | 深褐色 `#4A3F35` | 可读性 |

**弹出菜单配色：**

| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | 象牙白 `#FFFEF9` | 移除黑底 |
| 选中项背景 | 米色 `#FDF6ED` | 当前选中 |
| hover 背景 | 浅米色 `#FDF8F0` | 交互反馈 |

**实现方式：**
- 修改 `chat_panel.py` 中 `QComboBox` 的样式表
- 使用 QSS 自定义 `QComboBox::drop-down` 和 `QComboBox QAbstractItemView`

## 文件修改

| 文件 | 修改内容 |
|------|----------|
| `app/widgets/sidebar.py` | 重写 `drawBranches()` 方法，绘制三角形 |
| `app/widgets/chat_panel.py` | 添加 `QComboBox` 样式表 |

## 验收标准

1. 展开/闭合按钮无黑底，三角形清晰可见
2. 下拉框弹出的列表无黑底，背景为象牙白
3. hover 状态有正确的颜色反馈
4. 整体风格与应用"温暖编辑风格"协调一致
